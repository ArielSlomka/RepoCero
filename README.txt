Para levantar este repo en tu computadora tenes que:
1 - ir a la carpeta de tu repo en tu compu y abrirla con VSC.
2 - abrir la terminal en el directorio de esa carpeta, chequear que estas en la rama 'main' y ejecutar 'git pull'.
3 - luego ejecutar 'npm install'.
4 - luego de instalar las dependencias, ejecutar npm run dev para compilar sass en tiempo real.
